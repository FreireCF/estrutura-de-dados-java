## 🚀 Como compilar

Você pode compilar qualquer arquivo `.java` com o `javac`:

```bash
javac pilhas/PilhaVetor.java
java pilhas.PilhaVetor
```

Se preferir, use o Maven ou Gradle (em breve no projeto).
